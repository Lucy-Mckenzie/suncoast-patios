## Installation
1. Clone the repository:
```bash
 git clone git@github.com:Lucy-Mckenzie/suncoast-patios.git
```
2. Navigate to project directory:
```bash
cd suncoast-patios
```

3. Install dependencies:
```bash
 npm install
```

4. Start the deployment server:
```bash
npm run dev
```
